

PRÁCTICA 1 - PROCESADORES DE LENGUAJES -


Conversor de temperaturas (JavaScript)

- Se debe realizar un conversor de temperaturas de grados Celsius a Fahrenheit y Fahrenheit a Celsius.
- Realizar tests unitarios con mocha.js y chai.js.


- No se aceptan temperaturas como válidas si:
	-> No se especifica la unidad de temperatura.
	-> Se escribe más de un espacio en blanco entre el número y la unidad de temperatura.
	-> Se escribe cualquier caracter antes y/o después de la temperatura.


- Nota:
	-> Si se escribe un punto después de un número y no se especifíca número decimal, se considera como .0
	(Ejemplo: 12.c = 12.0c)
